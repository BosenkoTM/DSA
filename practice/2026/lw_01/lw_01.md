# Лабораторная работа №1. Реализация базовых структур данных (стек, очередь) на Python

## Описание
В рамках данной работы студентам предстоит изучить внутреннее устройство линейных структур данных. Основной упор делается на реализацию структур «с нуля» без использования высокоуровневых абстракций языка Python, а также на их применение для решения алгоритмических задач.

**Актуальные варианты заданий доступны на платформе:** [envlab.ru](https://envlab.ru/mod/assign/view.php?id=1037)

## Технологический стек
*   Python 3.10+
*   Git
*   Draw.io (или аналог для блок-схем)

## Цель работы
Изучить устройство, принципы работы и области применения Стека, Очереди и Дека. Научиться реализовывать эти структуры, используя базовые возможности языка (списки/массивы), и применять их на практике.

## Задачи
1.  Ознакомиться с теоретическим материалом.
2.  Реализовать класс структуры данных (Стек или Очередь) согласно варианту.
3.  Разработать алгоритм решения прикладной задачи с использованием реализованной структуры.
4.  Составить детализированную блок-схему алгоритма.
5.  Оформить репозиторий на GitHub/GitVerse (код, README, блок-схема).

## Критерии оценки (Максимум 10 баллов)
*   **Практическая реализация (5 баллов).** Корректность кода, соответствие PEP8, качество блок-схемы, оформление репозитория.
*   **Тестирование (5 баллов).** Прохождение автоматического теста на платформе Moodle для подтверждения теоретических знаний.
