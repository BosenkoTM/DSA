# Лабораторная работа №5. Реализация простого алгоритма машинного обучения

## Описание
Работа посвящена изучению «внутренней кухни» алгоритмов Machine Learning. Студентам необходимо реализовать математические модели (например, регрессию, k-NN, k-Means) «с нуля», используя низкоуровневые операции, без применения готовых методов обучения из библиотек типа scikit-learn.

**Актуальные варианты заданий доступны на платформе:** [envlab.ru](https://envlab.ru/mod/assign/view.php?id=1099)

## Технологический стек
*   Python 3.10+
*   Библиотеки: `numpy`, `pandas`, `matplotlib` (для визуализации)
*   Git, Draw.io

## Цель работы
Понять принципы работы алгоритмов ML и предварительной обработки данных через их программную реализацию с использованием матричных операций (NumPy).

## Задачи
1.  Изучить теоретические основы выбранного алгоритма.
2.  Реализовать класс ML-модели или инструмента препроцессинга (методы `fit`, `predict` и др. пишутся самостоятельно).
3.  Разработать скрипт для демонстрации работы модели на данных.
4.  Составить блок-схему ключевого метода обучения или предсказания.
5.  Оформить репозиторий с кодом, графиками и документацией.

## Критерии оценки (Максимум 10 баллов)
*   **4 балла.** Корректная реализация класса алгоритма на Python/NumPy (без import sklearn for training).
*   **4 балла.** Детализированная блок-схема алгоритма.
*   **2 балла.** Качественное оформление репозитория и README.
